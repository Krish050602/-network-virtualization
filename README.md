# Network Virtualization in Cloud Computing

## 🌐 Project Overview
This project demonstrates Network Virtualization concepts used in modern cloud computing platforms like AWS, Azure, and Google Cloud. It simulates a multi-tenant environment where multiple customers can create isolated virtual networks with their own IP address ranges, subnets, and virtual machines.

## 🎯 Key Concepts Demonstrated
- **Virtual Private Clouds (VPCs)** - Isolated virtual networks
- **Subnets** - Network segmentation within VPCs
- **Multi-Tenancy** - Multiple isolated tenants on shared infrastructure
- **IP Address Management** - Automatic IP allocation from CIDR blocks
- **Network Isolation** - Complete separation between tenant networks
- **Virtual Machine Management** - Create, start, stop VMs
